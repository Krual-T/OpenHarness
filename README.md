# OpenHarness

OpenHarness is a Codex-oriented skills/workflow repository derived from parts of [`obra/superpowers`](https://github.com/obra/superpowers) and adapted for the OpenRelay/OpenHarness repository workflow.

## What this repo contains

- `.codex/skills/`: the copied and evolving skills library from `openrelay`
- `AGENTS.md`: the repository map and workflow contract used by OpenHarness-style repos
- `.codex/INSTALL.md`: Codex installation instructions in the same "Fetch and follow instructions from ..." style used by `superpowers`

## Installation for Codex

Tell Codex:

```text
Fetch and follow instructions from https://raw.githubusercontent.com/Krual-T/OpenHarness/refs/heads/main/.codex/INSTALL.md
```
