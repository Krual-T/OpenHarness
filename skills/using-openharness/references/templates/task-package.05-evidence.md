# Evidence

## Residual Risks
- 用中文写出做完这一轮后仍然存在的风险、盲区或未覆盖面。

## Manual Steps
- 用中文记录需要人工执行的补充步骤；如果没有，就明确写无。

## Files
- 用英文路径列出本轮变更或关键证据涉及的文件；如果重要，顺手说明它支撑什么结论。

## Commands
- 用英文命令列出本轮实际执行过的命令，并标出 `final verification command`。

## Artifact Paths
- 用英文路径列出验证产物、日志或其他证据位置；如果没有，就明确写无。

## Follow-ups
- 用中文写后续任务、剩余决策或延后事项。
